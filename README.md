🔗 Live Demo

📌 Travel Landing Page – OIBSIP Level 1 Task 1

A modern and responsive Travel Landing Page built using HTML and CSS.
This project focuses on layout design, responsiveness, and clean UI structure.


🚀 Features
Full-screen Hero Section with overlay text
Interactive Search Bar UI
Popular Destinations grid with cards
Special Offers section with call-to-action
Testimonials with ratings and user info
Statistics section (ratings, users, destinations)
Fully responsive design (mobile + desktop)
Modern UI with shadows, spacing, and typography


🛠️ Tech Stack
HTML5
CSS3 (Flexbox + Grid)
Font Awesome (icons)
